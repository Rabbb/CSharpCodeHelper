# CSharpCodeHelper
C sharp code helper. class method extends.


## 项目代码分类

- 常用类型的扩展方法
  - DateTimeHelper 时间类
  - DictionaryHelper 字典类
  - EnumerableHelper 枚举类
  - JObjectHelper JSON读写(Newtonsoft.Json)
  - ListHelper 列表方法
  - MathHelper 数学计算
  - NumberHelper 数字处理
- 企业微信审批的回调数据解析
  - 审批表单读取代码
  - 测试代码实例
